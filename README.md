## Download & Setup

### Step 1: Download the Project Files

Download all 4 zip files from this repository:
- `AJ_Project.zip`
- `AJ_Project.z01`
- `AJ_Project.z02`
- `AJ_Project.z03`

**Important:** All 4 files must be in the same folder (e.g., Downloads).

### Step 2: Extract the Files

Use **[The Unarchiver](https://apps.apple.com/app/the-unarchiver/id425424353)** (free on the Mac App Store) to extract the files:

1. Install "The Unarchiver" from the Mac App Store
2. Right-click on `AJ_Project.zip` (the main .zip file, not .z01/.z02/.z03)
3. Select **Open With** â†’ **The Unarchiver**
4. The app will automatically combine all parts and extract the `AJ` folder

### Step 3: Run the Application

After extraction, read the `SETUP.md` file inside the `AJ` folder for instructions:

```bash
cd ~/Downloads/AJ
./start.sh
```

Then open **http://localhost:5173** in your browser.
